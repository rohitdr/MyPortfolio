## ⚙️ Tech Stack

* 🌐 Next.js (React Framework)
* 🎨 Tailwind CSS
* 🎞️ Framer Motion



## ✨ Features

* 🧑‍💻 Project showcase with detailed pages
* 🎥 Demo video integration
* 📊 Architecture & system design sections
* 🎨 Modern UI with animations
* 📱 Fully responsive design
* 🔗 External project linking (GitHub, live demos)


## 🧠 Key Concepts

* Component-based UI architecture
* Smooth animations using Framer Motion
* Clean project presentation & storytelling
* Performance-focused design



## 📁 Folder Structure

```
src/
├── app/                     # Next.js App Router
│   ├── page.jsx             # Home page
│   ├── chatnova/            # ChatNova project page
│   ├── reciperiot/          # RecipeRiot project page
│   └── layout.jsx            # Root layout
│   └── components/           # Reusable UI components
│       ├── project/          # project page-specific components
│  

```
